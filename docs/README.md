## Getting Started

- Install dependencies: `npm install`
- Run locally: `npm run dev`
- Build client: `npm run build`

For deployment guides, see `DEPLOYMENT.md`, `CUSTOM_DOMAIN_SETUP.md`, and `POCKETBASE_SETUP.md` at the repo root.
